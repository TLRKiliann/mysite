JavaScript

Au fur-et-à-mesure de mon apprentissage dans JavaScript, j'ai tenté de découvrir touts les domaines se rattachant à ce language. Il est vrai que de passer à Python3 à JavaScript n'a pas été un passage facile, principalement à cause de la synthax des functions implémentées dans JavaScript.

Il n'en demeure pas moins que je suis resté motivé pour apprendre et cela s'explique par mon intérêt porté au fonctionnement d'un navigateur, le moteur d'un navigateur, le DOM, le rendu avec JavaScript et le "frame-work" React (virtual DOM).

J'ai encore pour object d'apprendre TypeScript.
