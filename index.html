About me

Passioné par l'informatique, j'ai cherché à comprendre comment fonctionne un ordinateurs, internent (LAN et WAN), un routeur, les échanges serveur-client, les framework, les application. Pour cela, je me suis aidé de virtualisation (virtualbox) pour tester avec wireshark, scapy, nmap, gobuster, reverse-shell, etc.
Je me suis orienté vers le web développement et plus précisément dans le domaine du Front End Web Developer (React-Redux). J'aime également le Software Programming avec python3 en environnement virtuel avec anaconda-jupyter ou avec virtualenv.

Au niveau du hardware, je me suis également intéressé à comprendre comment optimiser l'usage des ressources. Savoir où regarder pour voir les nice, la mémoire virtuelle, l'espace d'échange (swap), sur web browser également avec inspect, page source, console et tout l'attirail qui s'accompagne de la touche F12, en plus de devtool react dev tool.
Comment un MMU gère les adresses physique et virtuelle avec la RAM et le CPU. Comment un PID est créé et comment l'arrêter en cas de nécessité (de manière non violente).

J'accorde une grande importance à la sécurité que ce soit au niveau de l'OS, infrastructures, connexion internet, des droits utilisateurs sur les fichiers, les ports utilisés par les applications, login, extensions, environements virtuels, les updates, debuggage, testing, etc.
